## 📖 Sobre o Projeto

Este projeto foi desenvolvido como **Trabalho de Conclusão de Curso (TCC)** da disciplina de **Sistemas Embarcados**.

O objetivo foi criar um **Cofre Inteligente**, utilizando Arduino, capaz de controlar a abertura e o fechamento do compartimento por meio de servo motores, além de fornecer uma interface simples para interação do usuário através de LEDs, botões e display LCD.

Durante o desenvolvimento foram aplicados conceitos de:

- Sistemas Embarcados
- Programação em C/C++
- Automação
- Eletrônica
- Integração entre Hardware e Software

## 🚀 Como Executar

1. Clone este repositório.

```
git clone https://github.com/SEU-USUARIO/SmartSafe-Arduino.git
```

2. Abra o projeto na Arduino IDE.

3. Instale a biblioteca:

```
Servo.h
```

4. Faça o upload para o Arduino Uno.

5. Conecte todos os componentes conforme o circuito desenvolvido.
